ComidApp
Planificador de menús y recetas · 2024
ComidApp es una plataforma web desarrollada en .NET Core y Angular que ayuda a las personas a planificar su alimentación semanal, generar listas de la compra automáticas y descubrir nuevas recetas personalizadas.

El proyecto está pensado para ser práctico, intuitivo y visualmente agradable, combinando desarrollo backend robusto, base de datos PostgreSQL y un frontend moderno.
  • Gestión y recomendación inteligente de recetas.
  • Creación automática de listas de la compra.
  • Panel de usuario personalizable.
  • API REST en .NET Core y base de datos en Supabase.
  • UI hecha a medida en Angular, mobile first.
